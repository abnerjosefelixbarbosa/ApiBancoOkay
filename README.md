# Descrição 

Essa API, criada na data de 15/02/2023, tem como objetivo gerenciar serviços para a aplicação do banco okay.

# Features

- Java 17
- JPA hibernate
- Maven
- Banco de dados H2 
- Spring boot
- Lombok
- Postman
- Validation
